## ЧТО НУЖНО ДОРАБОТАТЬ:
- При выборе заполненной ячейки с цифрой, заметки с тем же значением должны выделиться жирным шрифтом (ИМЕННО ЧИСЛО, ЧТО ВЫБРАНО)
- Кнопки сделать побольше, улучшить корректное отображение ячеек и кнопок, избегая обрезания.
- Темную тему

## Замечания:
- Кнопка становится недоступной, если поставлено определенное кол-во чисел. Если последнее число поставлено неправильно, кнопка станет недоступной.

## ЧТО В БУДУЩЕМ МОЖНО ДОБАВИТЬ:
- Аккаунты
- Совместную игру (соревнование в реальном времени)
